# Lista de Exercícios - Java

Esse é um programa feito em Java com um menu de exercícios. Foi feito como atividade da faculdade no primeiro semestre.

## O que o programa faz

O programa mostra um menu com 4 opções e fica rodando até o usuário digitar 0 para sair.

## Opções do menu

**1 - Desenho**
Só printa um desenho de um rosto no terminal, foi o primeiro exercício.

**2 - Calcular Distância**
O usuário digita a latitude e longitude de dois pontos e o programa calcula a distância entre eles em km. Usei uma fórmula matemática com seno e cosseno pra isso.

**3 - Contar Letras, Espaços e Caracteres**
O usuário digita um texto e o programa conta quantas letras, números, espaços e outros caracteres tem no texto. Usei um for pra percorrer cada caractere.

**4 - Múltipla Escolha**
Uma pergunta sobre a matéria de IHC (Interação Humano-Computador). O usuário tem até 3 tentativas pra acertar. Se acertar aparece em qual tentativa acertou, se errar as 3 aparece uma mensagem dizendo isso.

## Como rodar

Precisa ter o Java instalado. Depois é só compilar e rodar:

```
javac lista_exercicios.java
java lista_exercicios
```

## O que aprendi fazendo isso

- Usar o Scanner pra ler o que o usuário digita
- Usar switch pra fazer o menu
- Usar do-while pra ficar repetindo até o usuário sair
- Trabalhar com char pra verificar cada letra de um texto
- Usar funções (métodos) pra organizar o código
